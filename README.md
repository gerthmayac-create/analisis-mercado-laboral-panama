# 📊 Análisis del mercado laboral en Panamá

## 📌 Objetivo
Analizar la estructura del mercado laboral panameño considerando salario, informalidad y participación laboral.

## 🛠️ Herramientas
- Python
- Pandas
- Matplotlib

## 📊 Análisis realizados
- Brecha salarial por sexo
- Distribución de salarios
- Nivel de informalidad
- Análisis de población no económicamente activa

## 🔍 Principales hallazgos
- Existencia de brechas salariales por género
- Alta informalidad en ciertos grupos
- Diferencias estructurales en participación laboral

## 📁 Dataset
Datos provenientes de fuentes oficiales de Panamá.
